# 🚖 Uber Analytics Dashboard 

A complete **end-to-end Power BI project** that analyzes Uber trip data to uncover insights on bookings, revenue, distance, and customer behavior. This dashboard demonstrates strong **data visualization, DAX, and business intelligence skills**.

---

## 📊 Project Overview

This project focuses on building an interactive **Uber Analytics Dashboard** using Power BI. It provides key insights into:

- Completed bookings trends  
- Lost bookings analysis  
- Revenue performance  
- Distance traveled  
- Customer & driver ratings  

---

## 🎯 Objectives

- Analyze ride booking patterns over time  
- Identify factors contributing to lost bookings  
- Track revenue and trip performance  
- Evaluate customer and driver satisfaction  
- Build a visually appealing and interactive dashboard  

---

## 🧰 Tools & Technologies

- **Power BI**
- **DAX (Data Analysis Expressions)**
- **Power Query**
- **Data Modeling**

## 📈 Dashboard Insights

- Peak bookings observed mid-year (June–July)  
- Noticeable fluctuations in monthly demand  
- Revenue remains relatively stable despite booking variations  
- Lost bookings indicate potential operational inefficiencies
